## React App

### 配置 IDE（vscode）

在 settings.json 中增加以下配置：

```
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  },
  "files.autoSave": "onFocusChange",
  "editor.formatOnType": true,
  "editor.formatOnSave": true,
  "eslint.enable": true
```

安装 vscode 插件：

- ESlint
- Prettier - Code formatter

### 安装依赖

```
$ yarn
```

### 启动

```
$ yarn start
```
